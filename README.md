# Smart Resume Screening & Ranking System



A modular recruitment system designed to automate resume screening and candidate ranking based on job requirements.

## 🏗️ Architecture 



This project is based on **Microkernel Architecture**, consisting of a core system and multiple independent plugins.



## 🔌 Plugins






- 🔔 **Notification** – Manages system notifications.

- 📊 **Analytics** – Provides recruitment-related insights.



- 📄 **Timeline** – Extracts time of parsing, shortlisting, and ranking.

## 🎨 Design Patterns



- **Singleton Pattern** – Used for centralized `PluginManager`.

- **Strategy Pattern** – Used to support different candidate ranking strategies.



## ⚙️ System Workflow



1. Candidate resume is uploaded.

2. Resume information is extracted.

3. Candidate is screened according to job requirements.

4. Eligible candidates are ranked.

5. Notifications and analytics are generated.



## 🌟 Key Features



- Modular plugin-based system

- Automated resume screening

- Candidate ranking

- Job requirement matching

- Resume parsing

- Recruitment analytics

- Notification management

- Extensible and maintainable architecture



## 🎯 Objective



To develop a smart and modular system that reduces manual recruitment effort and helps identify the most suitable candidates efficiently.
